Head Snippets

What are these <head> snippets?
File Why
favicon.ico Legacy browsers and desktop tabs still default to this file. It's often auto-requested by browsers.
favicon-32x32.png Modern browsers use this when available, especially for desktop tabs with high-res screens.
apple-touch-icon.png Required for iOS homescreen icons when users "Add to Home Screen".
android-chrome-*.png Android browsers and PWAs use these for homescreen icons, recent apps thumbnails, and shortcuts.
safari-pinned-tab.svg Used only for Safari pinned tabs; SVG format allows theming and scaling.
site.webmanifest Required for PWA support, defining icons, names, colours, and behaviour on Android and modern browsers.
mstile-150x150.png Used by Windows 10/11 tiles (if user pins site).
<meta name="theme-color"> Defines browser toolbar colour on mobile devices and desktop Chrome.
